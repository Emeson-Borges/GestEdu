# GestEdu

GestEdu é um sistema de software de gestão escolar projetado para facilitar a administração escolar.

## Funcionalidades
- Gestão de alunos
- Gestão de professores
- Gestão de turmas
- Relatórios e análises

## Instalação
```bash
git clone https://github.com/usuario/GestEdu.git
cd GestEdu
npm install
```
